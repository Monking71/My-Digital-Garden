A proposition is a declarative statement - objectively true or false, not both.
Variables in propositional logic must be constant - e.g. cannot use time or place.
T > True
F > False
### Connectives
- 'not' | $\neg$ 
- 'and' 
- 'or'
- 'if... then'
- 'if and only if'
